3. 부모자식 태그

ex) parent
      child /child
     parent

- 포함하고 있는 태그를 부모 태그, 포함된 태그를 자식 태그
- 부모자식 관계가 바뀌지만, 항상 붙어다니는 사이 좋은 태그들이 있음.

  • 정다희
  • 희다정
  • 정희다

- li = 목차, ul(unordered list) = li의 부모 태그
- li 같은 항목들은 항목들끼리 그룹ing을 위한 부모 태그를 가짐.
- ol(ordered list): 앞에 자동으로 숫자 넘버링됨.
