6. 웹페이지 완성!

- href="" 이용해서 목차에 링크 연결하기
- Duplicate(복제)해서 각각의 링크가 가리킬 파일들(1.html,2.html...) 만들기
- 파일들 각각에 맞게 내용들 수정하기

- 웹 호스팅(깃허브 Github) 이용해 웹 서버 만들기
  -> new repository(저장소) 클릭해서 웹서버 생성
  -> Repository name 정하고 Public이랑 Initialize this repository with a RENAME 누르기
  -> upload files 누르고 파일 업로드
  -> commit changes에 정보 써주고 밑에 버튼 누르면 보관됨
  -> 파일 수정 시, 수정한 파일 업로드 하고 commit changes에 수정된 정보 써주고 버튼 누르면 됨.
  -> settings -> Github Pages
  -> Source 밑에 None을 main-/root로 바꾸고 저장
  -> 그러면 우리가 만든 웹사이트의 주소를 보여줌.

내가 만든 웹페이지 주소: https://heedaee.github.io/WEB-1/.
