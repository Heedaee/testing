- 만약 웹사이트를 아름답게 하고 싶다면, CSS를 배워야 함.
- 만약 사용자와 상호작용하는 웹페이지를 만들고 싶다면, JavaScript를 공부해야 함.
- 만약 생산성의 한계를 극복하고 싶다면, back end 기술을 배워야 함.
  -> 하나의 파일을 변경하면 1억개 웹페이지가 동시에 바뀌는 폭발적인 효과를 경험할 수 있음.
  -> JSP, PHP, Node.js etc.
    (이들은 서로 경쟁관계라서, 이들 중 하나를 공부하면 됨.)
- 만약 웹페이지에 광고를 달아 수익 창출하고 싶다면, Advertisement 수업을 들으면 됨.
